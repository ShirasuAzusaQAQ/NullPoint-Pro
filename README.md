# NullPoint-Pro
NullPoint-Pro is a Crystalpvp hack for Minecraft anarchy servers.
> [!IMPORTANT]
> This version is an additional update by volunteers. Since the original author seems to have stopped developing this project, this version was developed by volunteers.

## Developer compilation
```bash
./gradlew build
```

> [!TIP]
> This mod only works with Minecraft Fabric 1.20.4.

## Other
